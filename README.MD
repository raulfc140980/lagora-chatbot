# Agora Assistant

Agora Assistant is an AI-powered conversational assistant designed for the Agora College intranet environment.

The platform centralizes academic and administrative assistance for students, teachers, and administrative staff through a modern conversational interface integrated into the college intranet system.

---

# Project Purpose

The purpose of Agora Assistant is to improve the digital experience within the college environment by providing:
- intelligent conversational assistance;
- centralized access to information;
- automated FAQ support;
- academic and administrative guidance;
- and modern AI-driven intranet services.

The assistant helps users quickly access:
- schedules;
- exams;
- academic resources;
- administrative procedures;
- internal documentation;
- and appointment-related support.
